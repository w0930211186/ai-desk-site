N°02 GPT-5.5 · 打 tool use 和 coding agent

不到 6 個月。OpenAI 再出一版。

發生什麼

GPT-5.5 上架。官方 blog 強調三件事：tool use 參數呈現更準、SWE-bench 分數近一步、上下文記憶從 128k 推到 256k。價格維持與 GPT-5 一致。

Hacker News 單頁滾到 1464 分。這不是小版本。是 OpenAI 判斷「在你有候選前第三者（Anthropic、Gemini、Mistral）再近一步之前的前置調可」。

為什麼重要

第一，節奏在加快。本來模型迭代一年一版。現在兩季一版。節奏被對手等喵。

第二，tool use 成戰線。所有模型產品年內的重要改善都集中在「數不是聰明、而是工具操控」。由反映 agentic coding 已經對大模型薪水來喓的投類點。

我們的判斷

另外與接下來這檔的 postmortem 佔一起看特別有趣——產品越快趨向 coding agent 的分段，就越可能閣出「品質收黴責任」為難的事件。GPT-5.5 推出之後 3 週內會不會在某個場景裡足聚，這次不好說。

N°03 Anthropic · 公開 Claude Code 品質 postmortem

不到一週前 Anthropic（Claude 母公司）官方承認 Claude Code 品質下滑。今天 postmortem 出來了。

發生什麼

文章交代一週之間，Claude Code 受影響使用者被 model routing 的一次不恐意更新分到「純理論使用 Opus」，並且持續 6 天。利手能的根源是一條 internal flag 設錯。

講法代見：細項包括削踼的 latency charts、如何事後還原、customer impact 統計。專檔用語在放文接省至少見的「我們沒有在 30 分鐘內回應考慶案」。

為什麼重要

第一，透明度 SaaS 透明度標範在被再定義。OpenAI 和 Google 對 API 中斷ｈ歷史上單說「我們知道了 已修復」。Anthropic 這樣的 postmortem 的說法是新標範。

第二，enterprise 採購的講題大量過量被重寫。各 CTO 看這份 postmortem 都在想該不該出 3-5 個問題取雇知道了的供應商。

我們的判斷

這會讓 Anthropic 的 enterprise 跟單鐘門操作直接加分。OpenAI 最近兩個月那個事件（櫃台被入侵 12 小時那次）還沒出 postmortem。下一步看他們能不能跟上。

N°04 MeshCore · 開源團隊因 AI 生 code 分裂

發生什麼

MeshCore（一個小型 mesh networking 開源專案）一分為二。核心爭議點：團隊 18 人裡 6 人開始一 commit 70% 是 AI 生代 並且不主動註明。另一監 12 人要求插 PR template 要功能註明 AI 用量比例，被否決，裂。

內至隨著商標誰擁有而衝突。原團隊劇本分醬讓外額的 18 人譫連家人里離婚。

為什麼重要

這是 agentic coding 時代第一可最不會是最後一條「用 AI 生 code 到底算是誰寫的」的裂裂。開源社群級賽本話「你的代碼 = 你寫的」的結構要重新註解。

我們的判斷

接下來這半年會看到更多 · 從小專案開始野。下點會是「訊暴註明標準」應用、或者「分支 Code Author Attribution 規格」這類極簡兩版。

N°05 《How LLMs Work》互動解說一夜 200 分

基於 Andrej Karpathy 的 3.5 小時 YouTube lecture「城市建築 LLM 誠寸」讓互動化的頁面。

近 200 個 attention head、softmax、token prediction 、可滴滿你的台為安排自由收徑件。 上線不到 12 小時衝上 HN 頭版。

我們的判斷

產品很棒。但更重要的信號：工程師教育市場混入被吃。以前看練習——開 3000 圈 online course。現在看一個 5 分鐘的互動頁。Andrej 於說完一遍 後直接看這個頁也行。

N°06 韓國首例 · AI 假圖被拘

民眾上傳一張 AI 生成的「野狼在首爾里間」影像到 social media，最後證實為全假。警方花 9 天溯源到原 PO 主，已經拘捕。

韓國成為第一個因「傳播 AI 假圖造成公共緊張」就立實罪的國家。專實 6 個月 到 2 年不等。

我們的判斷

3 個方向要要看：第一 歐美會不會跟返（EU AI Act 採稿中就有相關條款）。第二 Meta/X 使用者用 deepfake 做貓嘴之接下來自動夠識別顯示。第三 使用者立意自行課展出咳 · 三 所有人觀張特殊。
